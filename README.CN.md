## 🌊 关于这个作品集

这个仓库用于展示我的个人学术与职业主页：

### 🔗 [访问我的网站 → love-ovo.github.io](https://love-ovo.github.io/)

你好，我是 **王俞亮**，目前是 **南洋理工大学 Maritime Studies** 硕士学生。

我的背景结合了 **海事运营**、**货运代理**、**航运数据分析**、**物流** 和 **供应链研究**。这个作品集展示了我的教育背景、研究兴趣、代表性发表、技术能力和职业方向。

> **海事知识 + 数据分析 + 运营洞察**

## 📊 从数据到决策的流程

| 阶段 | 重点 | 输出 |
|---|---|---|
| **01 · 海事情境** | 航运、港口、物流、安全和运营问题 | 明确的研究或运营问题 |
| **02 · 数据收集** | 海事、港口、物流或交通相关数据 | 结构化数据来源 |
| **03 · 处理与分析** | Python、MATLAB、SPSS、GIS、Excel 和统计方法 | 清洗后且可解释的数据集 |
| **04 · 建模与预测** | 预测、优化和定量模型 | 分析结果 |
| **05 · 运营洞察** | 将分析转化为建议 | 港口、航运或供应链决策 |
